# Dynamic To-Do List

A clean, interactive to-do list web application built with vanilla JavaScript. This project focuses on dynamic DOM manipulation, allowing users to seamlessly add, edit, and remove tasks without reloading the page.

 **Live Demo:** [View the app here](https://todolist-raquib.netlify.app/)

##  Features

* **Add Tasks:** Quickly add new items to your list.
* **Edit Tasks:** Built-in edit button swaps the task text for an input field, allowing quick updates.
* **Remove Tasks:** One-click deletion of completed or unwanted tasks.
* **Input Validation:** Prevents users from adding empty tasks with a custom alert.
* **Responsive Layout:** Flexbox-based design keeps the application centered and easy to use on different screen sizes.

##  Technologies Used

* **HTML5:** Semantic structure.
* **CSS3:** Custom styling, color themes, and Flexbox layout for neat alignment.
* **JavaScript :** Event listeners, dynamic element creation (`createElement`), and DOM node management (`insertBefore`, `removeChild`).
